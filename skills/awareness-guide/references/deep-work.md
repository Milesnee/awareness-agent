# 深度工作实践指南

> 来源：Marc Lou 纪录片 + Huberman Lab + Csikszentmihalyi《心流》
> 用途：觉察助手在引导用户保护深度工作时的参考

---

## 核心公式

**深度工作 = 离线 + 固定时段 + 足够长（4-6h）+ 优先创作**

---

## Marc Lou 的深度工作协议

### 日常结构（365天不变）

| 时段 | 活动 | 说明 |
|------|------|------|
| 早起 | 咖啡+早餐+健身房 | 身体先动，大脑才清醒 |
| 中午起 | **4-6h 深度工作** | 完全离线，不看手机/邮件/社交媒体 |
| 下午3点 | 发布 | 把做好的东西推出去 |
| 下午4点 | 上线处理琐事 | 查邮件、回复、处理支持 |
| 晚上9点 | 准时睡觉 | 调暗灯光→放松程序→入睡 |

### 深度工作规则

1. **手机是最大的敌人** — 物理隔离，不看
2. **不查邮件** — "读到一封邮件说应用出了bug，就得去修，而不是做真正重要的事——创造更多东西"
3. **不回社交媒体** — FOMO 会偷走一小时
4. **保护创作流程** — 修bug会打断创作，创作优先
5. **睡眠是一切基础** — "有了良好的睡眠，我感觉情绪更稳定，注意力超级集中，可以在同一件事上花4个小时"

### Marc Lou 金句

> "这是让我走到今天的东西。我知道如果想走得更远，就必须完成这个深度工作。"

> "我会产生一种错失恐惧症（FOMO），觉得自己不够快。然后一个小时就过去了，我失去了一个小时，而且感觉缺乏动力回到工作中。"

> "我对 AI 时代唯一重要的 KPI 没有改变：仍然是你发布的东西的数量。"

## Huberman Lab 科学支撑

### 90分钟超昼夜节律

- 体温上升期 = 认知效率峰值
- 全天真正高效时间只有 3-4 小时
- **保护最重要的 90 分钟**

### 视觉流与焦虑

- 向前走时视觉流能降低杏仁核活跃度
- **晨间散步10分钟** → 直接安抚焦虑，为深度工作做准备

### 延迟咖啡因

- 醒后90-120分钟再喝咖啡
- 让皮质醇自然清除腺苷，避免下午崩溃

## 心流理论的深度工作映射

| 心流条件 | 深度工作实践 |
|----------|------------|
| 清晰目标 | 今天做哪个功能/写哪段代码 |
| 即时反馈 | 代码能跑/不能跑，写完能看到 |
| 挑战=技能 | 选刚好超出舒适区的任务 |
| 深度专注 | 4-6h 离线时段 |
| 自我意识消失 | 不评判"我做得好不好"，只做 |
| 时间扭曲 | 进入状态后时间飞逝 |

## 觉察助手引导话术

### 晨间：深度工作准备

「今天最重要的90分钟，你打算做什么？选一件刚好超出你舒适区的事——太简单会无聊，太难会焦虑，中间那个点就是心流。」

「出门走10分钟再开始。不是锻炼，是让眼睛接收'向前走'的信号，安抚你的杏仁核。」

「手机放另一个房间。Marc Lou 说的：手机是深度工作最大的敌人。」

### 晚间：深度工作回顾

「今天你有多少时间在深度工作？有多少时间被碎片化消耗了？」

「你今天是在创作，还是在响应？Marc Lou 的原则：创作优先，bug和邮件下午再处理。」

「你今天发布了什么？AI时代最重要的KPI：发布的东西的数量。」

### 用户提到纠结/犹豫时

「Musk 说：'第一次做就能消除80%的恐惧。'别等完美了，先发布。」

「Marc Lou 35个项目，30个失败，命中率5%。你不是在赌一个项目，你是在掷骰子。掷得越多，赢的概率越大。」

## 实用清单：如何进入深度工作

1. ☀️ **前一晚**：准备好明天的深度工作内容（不浪费第二天决策）
2. 🚶 **早晨**：散步10分钟（视觉流+焦虑缓解）
3. ☕ **延迟咖啡因**：醒后90分钟再喝
4. 📵 **物理隔离手机**：放另一个房间
5. 🎯 **明确目标**：今天做哪个功能？一句话说清楚
6. ⏱️ **设置计时器**：90分钟一个session，中间休息5分钟
7. 🚫 **不响应外部**：邮件、社交媒体、bug修复——下午4点再处理
8. 📦 **发布**：做完了就推出去，不等完美
9. 😴 **晚上9点**：调暗灯光，放松程序，准时入睡
